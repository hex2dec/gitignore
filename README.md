# gitignore

This repository contains a small set of commonly used `.gitignore` templates for Bun, Go, Java, Node, Python, and Rust.

## Templates

- `bun.gitignore`
- `Go.gitignore`
- `Java.gitignore`
- `Node.gitignore`
- `Python.gitignore`
- `Rust.gitignore`

## Acknowledgments

These templates are sourced from the official [`github/gitignore`](https://github.com/github/gitignore) project. Thanks to the GitHub `gitignore` maintainers and contributors.
