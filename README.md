# gitignore

This repository contains a small set of commonly used `.gitignore` templates for Go, Java, Python, Rust, and TypeScript.

## Templates

- `Go.gitignore`
- `Java.gitignore`
- `Python.gitignore`
- `Rust.gitignore`
- `TypeScript.gitignore` (includes Node.js and Bun)

## Acknowledgments

These templates are sourced from the official [`github/gitignore`](https://github.com/github/gitignore) project. Thanks to the GitHub `gitignore` maintainers and contributors.
